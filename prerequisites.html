Prerequisites

Before starting this how-to, make sure you have completed the following prerequisites:

  • Create a new app or have an existing app project available.
  • Open the Mendix Development Portal in a web browser.
  • Open the app in edit mode in Mendix Studio. With Mendix Studio you can create and edit applications in your browser without installing software on your PC. For more information, see Mendix Studio General Info Page.
